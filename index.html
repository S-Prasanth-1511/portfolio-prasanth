Solution: A web application that uses a Convolutional Neural Network (CNN) to classify images as real or fake. The model focuses on analyzing subtle inconsistencies in the eye region, a common area for deepfake artifacts, using gradient-based analysis.

Secure EHR System with AI Anomaly Detection

Team Project

Problem: Traditional healthcare access controls are insufficient for secure, cross-organizational data sharing and lack robust insider threat detection.

Solution: A system combining Multi-Authority Attribute-Based Encryption (MA-ABE) for fine-grained access control with an AI engine that detects anomalous user behavior in real-time. The architecture uses a Flask backend, a cryptographic engine with the RW15 MA-ABE scheme, and an Isolation Forest model for anomaly detection.

Tech Stack: Python, Flask, SQLite, MA-ABE (charm-crypto), AES, scikit-learn.

View on GitHub →

Auditory Attention Decoding using EEG
